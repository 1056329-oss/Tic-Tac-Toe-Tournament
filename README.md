# Tic Tac Toe Tournament Web App

A fun and interactive tic-tac-toe tournament web application where players compete, get ranked, and track their statistics!

## 🌐 Live Demo

**Play Online:** https://tic-tac-toe-tournament-t79d.onrender.com/

## Features

✨ **Player Management**
- Register players with their names and ages
- Automatic player tracking and profile creation

🏆 **Ranking System**
- Players earn points based on match results:
  - **3 points** for a win
  - **1 point** for a draw
  - **0 points** for a loss
- Dynamic ranking leaderboard sorted by points and wins
- Win rate calculation

🎮 **Game Features**
- Play classic tic-tac-toe matches
- Real-time turn indicator
- Visual game status (winner/draw/current player)
- Multiple consecutive games possible

📊 **Statistics Tracking**
- Total matches played
- Wins, draws, and losses
- Win rate percentage
- Points accumulated

## How to Use

1. **Open the App**
   - Double-click `index.html` or open it in any modern web browser

2. **Navigate Pages**
   - Use the navigation buttons at the top: Home, Rankings, Play

3. **Start Playing**
   - Click "Start Playing" button
   - Enter both players' names and ages
   - Click "Start Game"

4. **Play the Game**
   - Players take turns clicking cells (X goes first)
   - First player to get 3 in a row wins
   - If all cells are filled with no winner, it's a draw

5. **View Rankings**
   - Click "Rankings" to see the current leaderboard
   - Rankings are sorted by points (3 for win, 1 for draw)
   - Medals: 🥇 1st, 🥈 2nd, 🥉 3rd

## Game Rules

- Standard tic-tac-toe rules apply
- Player 1 (X) always goes first
- Player 2 (O) goes second
- Match results are saved to local storage
- Rankings persist even after closing the browser

## Data Storage

- All player data is stored in your browser's local storage
- No data is sent to any server
- Your data stays private on your device
- Clear browser data to reset all rankings
