Project Description

This Tic Tac Toe Tournament is an interactive web application built to showcase skills in web development. The app features player registration, a dynamic ranking system, and persistent game statistics using browser storage.

Features:

  • Player management and profile tracking
  • Real-time ranking leaderboard
  • Statistics tracking (wins, losses, draws)
  • Classic Tic Tac Toe gameplay
